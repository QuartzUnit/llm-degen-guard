# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.1.0/),

## [0.1.0] - 2026-03-17

### Added
- 4-signal composite scoring engine (repetition, entropy, structure, semantic)
- Streaming and batch API for real-time and bulk analysis
- Structural discount to reduce false positives on structured outputs
- Zero external dependencies
- 55 tests with full coverage
